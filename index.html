About Me
Web Designing Is My Passion

Perkenalkan, nama saya Muhamad Yusuf, lahir 19 November 1994 di kota Cilegon Banten, umur 27 tahun dan masih lajang. Tahun 2015 saya masuk perguruan tinggi swasta tepatnya di Universitas Serang Raya (UNSERA) Kota Serang Banten. Kemudian pada tahun 2016 saya bergabung dalam organisasi Badan Ekseketif Mahasiswa (BEM) fakultas Teknologi Informasi UNSERA dan berbagi pengetahuan dengan rekan dan senior mengenai keorganisasian dan mengenai web programming dan web design sampai tahun 2019. Di luar aktivitas kampus, saya bekerja di salah satu yayasan kesenian sebagai instruktur seni di beberapa sekolah di kota Cilegon. Selama masa perkuliahan saya mengerjakan beberapa projek aplikasi, salah satunya aplikasi web E-learning untuk sekolah. Hingga pertengahan tahun 2022, tepatnya bulan juni lalu saya berhasil mendapat gelar sarjana Strata 1 jurusan Teknik Informatika. Saya menyukai web programming terutama design web pada bagian front end karna selaras dengan hobi saya, yaitu kesenian. So, Web Design Is My Passion.
